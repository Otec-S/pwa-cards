# Changelog

Все заметные изменения в проекте "PWA Карточки" будут документированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [семантического версионирования](https://semver.org/lang/ru/).

## [1.0.0] - 2026-01-21

### Добавлено
- Полная переработка проекта на TypeScript с типизацией
- Отображение карточек по одной на экране
- Навигация через кнопки-стрелки (влево/вправо)
- Навигация через свайпы (тач-события для мобильных устройств)
- Навигация через клавиши стрелок на клавиатуре
- Индикатор прогресса (текущая карточка / общее количество)
- Сохранение текущей позиции в localStorage
- Адаптивный дизайн для всех типов устройств
- Автоматическое масштабирование шрифта в зависимости от размера экрана
- Плавная анимация при смене карточек
- PWA функциональность с offline-поддержкой
- Service Worker для кэширования ресурсов
- Манифест PWA для установки на устройства
- Иконки приложения (192x192 и 512x512)
- Автоматическая сборка TypeScript с npm scripts
- Конфигурация TypeScript (tsconfig.json)

### Технические детали
- Использован TypeScript 5.3.0
- Модульная структура с разделением на app.ts, sw.ts, types.ts
- Хранение данных карточек в JSON формате
- Современный UI с градиентами и тенями
- Оптимизация для сенсорных устройств
